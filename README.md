# Telco Customer Churn Assessment

This repository contains my Worthwhile data scientist assessment submission using the public Telco Customer Churn dataset.

## Main Submission

- `Jyothirmai_Puram_Worthwhile_Assessment.pdf` - final report for easy viewing
- `Jyothirmai_Puram_Worthwhile_Assessment.md` - editable source version of the report

## Supporting Files

- `WA_Fn-UseC_-Telco-Customer-Churn.csv` - Telco Customer Churn dataset
- `churn_analysis.py` - full exploratory analysis and modeling workflow
- `generate_figures.py` - script used to generate report figures
- `figures/` - generated charts and model metric summary
- `top_products.sql` - SQL assessment query

## Reproducing the Analysis

Install the Python dependencies:

```bash
pip install -r requirements.txt
```

Run the analysis:

```bash
python churn_analysis.py
```

Regenerate report figures:

```bash
python generate_figures.py
```
